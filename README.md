# Игровой Компьютерный Клуб

Веб-сайт для игрового компьютерного клуба с возможностью бронирования, участия в турнирах и заказа еды из кафе.

## Описание проекта

Сайт состоит из трёх страниц:

- **Главная страница** (`glavnai.html`) - основная страница с информацией об услугах, галереей, новостями и контактами
- **Регистрация** (`regestrashi.html`) - форма регистрации новых пользователей
- **Турниры** (`tournament.html`) - расписание и информация о турнирах

## Функциональные возможности

- 🖥️ **Бронирование компьютеров** - возможность забронировать игровое место
- 🏆 **Турниры** - расписание турниров по CS:GO, Dota 2, FIFA
- ☕ **Кафе** - модальное окно с меню (напитки, основные блюда, закуски)
- 📰 **Новости** - слайдер с новостями клуба (используется Swiper)
- ❓ **FAQ** - часто задаваемые вопросы
- 📞 **Контакты** - форма обратной связи

## Технологии

- **HTML5** - структура страниц
- **CSS3** - стилизация и адаптивный дизайн
- **JavaScript** - интерактивные элементы
- **Swiper** - слайдер для новостей



## Адаптивность

Сайт полностью адаптивен будет скоро и корректно отображатся на:
- Desktop (1920px+)
- Ноутбуках (1024px - 1919px)
- Планшетах (768px - 1023px)
- Мобильных устройствах (320px - 767px)
